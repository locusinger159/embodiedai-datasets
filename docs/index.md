**EmbodiedAI Datasets** 专注收录全球**具身智能**、**人形机器人**、**机械臂**、**移动机器人**数据集情报。

### 收录标准

1. **真实性**: 只收录真实存在的数据集
2. **可用性**: 提供官方链接或申请入口
3. **完整性**: 包含基础字段信息（机构、规模、任务等）
4. **中立性**: 只做信息导航，不提供数据下载
